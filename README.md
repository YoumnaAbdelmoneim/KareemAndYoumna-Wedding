# Kareem & Youmna Wedding Invitation

## قبل الرفع على GitHub

1. افتح `index.html` وعدّل:
   - الأسماء.
   - التاريخ.
   - اسم القاعة والمكان.
   - رابط Google Maps.

2. افتح `script.js` وعدّل:
   - `weddingDate`
   - `whatsappNumber`

3. الصور:
   - ضع صورتكم داخل `assets` باسم `couple.jpg`.
   - في `index.html` غيّر:
     `assets/couple.svg`
     إلى:
     `assets/couple.jpg`
   - يمكنك وضع صورة غلاف باسم `cover.jpg` ثم تعديل `style.css` واستبدال:
     `assets/cover.svg`
     بـ:
     `assets/cover.jpg`

4. الموسيقى:
   - ضع ملف MP3 باسم `music.mp3` داخل `assets`.

## الرفع على GitHub

ارفع محتويات هذا الفولدر إلى الـRepository، وليس الفولدر الخارجي نفسه.

بعدها:
Settings → Pages → Deploy from a branch → main → /(root) → Save
